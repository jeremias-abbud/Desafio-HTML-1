Listas Ordenadas e Não Ordenadas

Para criar uma lista ordenada, utilizamos a TAG "ol". Para criar itens da lista, utilizamos a TAG "li". A lista será exibida com numeração.

Para criar uma lista não ordenada, utilizamos a TAG "ul". Para criar itens da lista, utilizamos a TAG "li". A lista será exibida com marcadores.

Podemos aninhar uma lista dentro da outra da maneira que for conveniente.
