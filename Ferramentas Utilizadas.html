Introdução ao HTML na prática

Ferramentas Utilizadas

Existem vários tipos de editores de HTML, mas um dos principais e mais utilizados é o , VS Code , que é gratuito e de fácil instalação.

Um plugin muito útil que pode ser instalado no VS Code é o "Live Server". Com ele, é possível visualizar em tempo real, no navegador, a página que está sendo criada no VS Code.

O site W3 Schools apresenta de forma organizada todas as funcionalidades que podem ser utilizadas em HTML.
