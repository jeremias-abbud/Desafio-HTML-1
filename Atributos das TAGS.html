Atributos das TAGS

Além dos nomes, as TAGS possuem atributos, que são propriedades globais ou específicas para que apareçam diferente no navegador. Os atributos podem receber os valores que modificam seu comportamento.

Alguns dos atributos mais utilizados são:

    id: cria uma identificação para o elemento. Esse atributo é muito utilizado em conjunto com o Java Script.
    style: serve para passar comandos de CSS para as TAGS, propiciando sua estilização no próprio HTML.
    class: cria uma classe padronizada para ser utilizada no CSS.
