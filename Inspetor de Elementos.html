Inspetor de Elementos

Os navegadores fornecem ferramentas de opções do desenvolvedor, que permitem visualizar a estrutura das páginas, possibilitando compreender como aquela página foi criada.

Para abrir o inspetor de elementos, podemos clicar F12 e selecionar a estrutura do elemento desejado. O elemento será exibido na página.

É possível visualizar tanto as informações HTML(marcação) quanto as de CSS (estilização) dos elementos dá página.
